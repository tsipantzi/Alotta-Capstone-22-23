# Alotta 2022-23 Liberty University Capstone Project

##### BUILD
```  
  In the main alotta folder run this command in the terminal
  This will also run the test suite beforehand once setup 
  ./gradlew clean build
```
##### TEST SUITE
```
  In the main alotta folder run this command in the terminal 
  ./gradlew test
```
##### RUNNING THE APP
```
  In the main alotta folder run this command in the terminal 
  ./gradlew clean bootRun
```
